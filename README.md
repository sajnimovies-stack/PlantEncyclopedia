# Plant Encyclopedia

Android app to identify plants using camera and AI inference.

## Features
- Capture plant images using camera
- (Future) AI-based plant identification
- Display plant info

## How to run
1. Clone this repo
2. Open in Android Studio
3. Build and run on your device

## Requirements
- Android Studio
- Android device or emulator
